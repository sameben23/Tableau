# Projects

This directory contains complete Tableau projects, dashboards, and visualizations.

## Structure

Each project should have its own folder with:
- Tableau workbook files (`.twb` or `.twbx`)
- Project README with description and documentation
- Screenshots or preview images (optional)
- Supporting files or data samples (if applicable)

## Example Project Structure

```
projects/
├── sales-dashboard/
│   ├── README.md
│   ├── sales-dashboard.twbx
│   ├── preview.png
│   └── data-notes.txt
└── customer-analysis/
    ├── README.md
    └── customer-analysis.twb
```

## Naming Conventions

- Use descriptive, lowercase folder names with hyphens
- Include dates for time-specific projects (e.g., `q4-2024-analysis`)
- Keep names concise but meaningful

## Adding a New Project

1. Create a new folder: `mkdir my-project-name`
2. Add your workbook files
3. Create a README.md documenting the project
4. Add any supporting materials
5. Commit with a descriptive message

See [CONTRIBUTING.md](../CONTRIBUTING.md) for detailed guidelines.
