# Assets

This directory contains supporting files and resources for Tableau projects.

## Purpose

Store reusable assets including:
- Images and logos
- Icons and shapes
- Backgrounds and textures
- Fonts (if redistributable)
- Custom color palettes
- Map files and geographic data

## Structure

```
assets/
├── images/
│   ├── logos/
│   └── backgrounds/
├── icons/
│   └── custom-shapes/
├── maps/
│   └── custom-geocoding/
└── colors/
    └── palettes/
```

## Asset Categories

### Images
- Company logos
- Background images
- Dashboard headers
- Preview thumbnails

### Icons and Shapes
- Custom shape files for Tableau
- Icon sets for dashboards
- Symbol libraries

### Maps
- Custom geographic data
- Geocoding files
- Spatial data

### Colors
- Color palette files
- Brand color specifications
- Accessibility-friendly palettes

## File Formats

- **Images**: PNG, JPG, SVG
- **Shapes**: Custom Tableau shape files
- **Maps**: Shapefiles, GeoJSON, KML
- **Colors**: XML, preferences files

## Best Practices

- Use high-quality images
- Optimize file sizes
- Name files descriptively
- Organize by category
- Include attribution for external assets
- Check licensing for third-party assets
- Use web-safe formats

## Image Guidelines

- Logos: PNG with transparent background
- Backgrounds: Optimize for dashboard size
- Icons: Consistent size and style
- Previews: 1200px width recommended

## Adding Assets

1. Choose the appropriate subfolder
2. Use descriptive filenames (lowercase-with-hyphens)
3. Optimize file sizes
4. Add a README if needed for attribution
5. Document usage instructions

## Attribution

If using third-party assets:
- Credit the source
- Verify licensing
- Include attribution in project documentation
- Store license info with assets
