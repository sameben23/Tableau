# Templates

This directory contains reusable Tableau dashboard templates and style guides.

## Purpose

Maintain consistent design and branding across Tableau projects with:
- Dashboard layout templates
- Color palettes and style guides
- Reusable chart configurations
- Standard formatting templates

## Contents

- **Dashboard Templates** - Pre-built layouts for common dashboard types
- **Style Guides** - Color schemes, fonts, and design standards
- **Chart Templates** - Standardized visualizations
- **Best Practices** - Design guidelines and recommendations

## Using Templates

1. Browse available templates
2. Download or copy the template workbook
3. Customize with your data and content
4. Follow the style guide for consistency

## Adding a Template

1. Create a template folder
2. Add your template workbook
3. Include a README with:
   - Template purpose and use cases
   - Customization instructions
   - Required data structure
   - Preview screenshots
4. Document any custom formatting or calculations

## Best Practices

- Keep templates generic and reusable
- Include clear instructions for customization
- Use placeholder data or parameters
- Document all custom formatting
- Include visual previews
